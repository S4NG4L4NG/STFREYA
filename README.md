<h2>Features</h2>

<ul>
  <li>
    <b>Student management</b>
  </li>
</ul>

<table>
  <tr>
    <th><b>Function</b></th>
    <th><b>Description</b></th>
  </tr>

  <tr>
    <td>
      <b>CRUD</b> (<i>Create, Read, Update, Delete</i>)
    </td>
    <td>
      <ul>
        <li>Users may add, update, and delete student entries.</li>
        <li>Users may search for a student through their name, email address, or contact number.</li>
        <li>Entry results can be filtered based on the enrolled course (<b>BSCS</b>, <b>BSIT</b>, or <b>BMMA</b>).</li>
      <ul>
    </td>
  </tr>
</table>

<ul>
  <li>
    <b>Student list</b>
  </li>
</ul>

<table>
  <tr>
    <th><b>Function</b></th>
    <th><b>Description</b></th>
  </tr>

  <tr>
    <td>
      <code>.CSV</code> <b>report generation</b>
    </td>
    <td>
      <ul>
        <li>A report modal can be viewed by the user, detailing overall statistics, students per course, course enrollment rate and distribution, as well as gender rate and distribution. This report may also be exported as a <code>.CSV</code> file.</li>
        <li>A student list aggregating every student entry present within the system can be generated by the user.</li>
        <li>An attendance checklist determined by the status each student entry carries (a student can be marked as <b>PRESENT</b>, <b>LATE</b>, OR <b>ABSENT</b>), may also be generated.</li>
        <li>A student list aggregating every student entry present within the system can be generated by the user.</li>
        <li>A downloadable performance list can be installed by the user, which curates every score that users have inputted per student.</li>
      <ul>
    </td>
  </tr>
</table>

<h3>Getting Started</h3>

<ul>
  <li><b>Running the project locally</b></li>
</ul>

<pre><code># Clone this repository
git clone https://github.com/S4NG4L4NG/STFREYA.git</code></pre>

<h3>XAMPP - Database account setup</h3>

<ol>
  <li>Open <b>XAMPP Control Panel</b> and start the <b>MySQL</b> service. </li>
  <li>Once the MySQL service has started, click <b>Admin</b> under <b>Actions</b>. You will then be redirected to <b>phpMyAdmin</b>.</li>
  <li>Under phpMyAdmin, go to <b>User accounts</b> > <b>New</b> > <b>Add user account</b>.</li>
  <li>Fill your credentials under the fields <b>User name</b>, <b>Password</b>, and <b>Re-type</b>.</li>
  <li>Check <b>Grant all privileges on wildcard on wildcard username (username_%).</b> and tick <b>Check all</b> under <b>Global privileges</b>.</li>
</ol>
